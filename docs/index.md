## Quick Start

### Installation

#### Homebrew (Recommended)

```bash
# Add the Tyk tap
brew tap sedkis/tyk

# Install the CLI
brew install tyk
```

### Initialize Configuration

```bash
# Run the interactive setup wizard
tyk init

# Quick setup for single environment
tyk init --quick
```
